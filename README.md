# Java internship technical test — session analysis

## Task

Implement logic that **calculates session time per user** from the provided event log.

The dataset is in `events.json`. Starter code is in `java/SessionAnalyzer.java` (adapt as needed).

## For candidates

1. Read `events.json` and the starter class.
2. Implement the required behaviour and any helpers you need.
3. Submit via **GitHub Classroom** using the invitation link from the recruiter (push to **your** assignment repository, not this template).

## Run (example)

```bash
cd java
javac SessionAnalyzer.java
java SessionAnalyzer
```

Adjust `main` or packaging if you add multiple files.
